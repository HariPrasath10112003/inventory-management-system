#  Inventory Management System

A full-stack Inventory Management System developed to manage products, track stock levels, and perform basic inventory operations efficiently.

---

##  Tech Stack

###  Frontend

* React.js
* HTML
* CSS
* JavaScript

###  Backend

* Java
* Spring Boot

###  Database

* PostgreSQL

---

##  Features

*  Simple Login (Company Email Based)
*  Add New Products
*  View All Products
*  Edit Product Details
*  Delete Products
*  Sell Products (Automatically reduces stock)
*  Low Stock & Out of Stock Identification
*  Dashboard with basic statistics
